Bootstrap Library

Welcome to the Bootstrap Essentials Library. This page serves as a quick reference guide to the key classes and components in Bootstrap. From utility classes for layout and spacing to core components. This guide is designed to make navigating Bootstrap’s features straightforward and efficient.

quick tips

Use Containers for Proper Layout

Always start with a .container or .container-fluid class to ensure proper alignment, padding, and responsiveness across various screen sizes.

Leverage the Grid System

Bootstrap’s grid system is fundamental for building layouts. Use .row and .col classes to create responsive, flexible column structures that automatically adjust for different screen sizes.

Customize with Buttons

Bootstrap provides a variety of button styles, such as .btn-primary and .btn-secondary. You can also modify the size with classes like .btn-lg or .btn-sm and use .btn-block for full-width buttons.

Take Advantage of Utility Classes

Bootstrap includes a wide range of utility classes like .mt-3, .mb-3, and .p-3 that let you quickly adjust margins, padding, and other spacing without writing custom CSS.

Use Breakpoints for Responsiveness

Bootstrap’s built-in breakpoints (e.g., .col-sm-4, .col-md-6) allow you to create layouts that adapt to different screen sizes, making your website mobile-friendly by default.

Customize with CSS Variables

If you want to change colors, spacing, or other design elements across your site, Bootstrap’s use of CSS variables allows for easy customization without modifying the core framework.

Incorporate Components for Faster Development

Bootstrap provides a variety of ready-to-use components, such as navbars, modals, and carousels. These can save you time while building a clean and professional UI.

Use Cards for Content

Cards are a great way to organize and display content in a structured way. They come with various styles like borders, background colors, and padding, making them highly customizable.

Grid System Mastery

Bootstrap’s grid system is based on flexbox and offers a flexible layout through rows and columns. Use .container or .container-fluid for a responsive layout, and remember to utilize .row and .col classes for alignment.

Responsive Typography

Use .display-1 through .display-6 classes for attention-grabbing headings and maintain consistency across devices with responsive text utilities like .fs-1 or .text-sm-start.

Spacing Utilities

Bootstrap has built-in margin (.m-#) and padding (.p-#) utility classes for fine-tuning spacing. Use .mt-3 for a top margin or .px-2 for horizontal padding. Just replace # with numbers from 0 to 5.

Form Customization

Leverage .form-control and .form-select classes for clean and functional forms. Enhance input fields further with .input-group for additional elements like buttons or text.

Responsive Embeds

Want to embed a video responsively? Use the .ratio utility class (e.g., .ratio-16x9) to ensure the video scales correctly across devices.

Navbars and Toggles

Create modern navigation bars with .navbar and use the .navbar-toggler in combination with the .collapse class to provide responsive toggle functionality.

Utility Classes for Visibility

Control element visibility with classes like .d-none, .d-md-block, or .d-sm-inline. These let you show/hide elements based on screen size breakpoints.

Alerts and Dismissible Elements

Create alerts using the .alert class with contextual colors (.alert-success, .alert-warning) and make them dismissible by adding the .alert-dismissible class with a close button.

Card Components

Use .card for creating flexible content containers. Combine .card-title, .card-body, and .card-footer for a polished, modern layout.

Buttons and Icons

Combine .btn classes with colors like .btn-primary or .btn-outline-secondary. Add icons from libraries such as Font Awesome or Bootstrap Icons for extra flair.

Buttons and Icons

Bootstrap’s built-in Flexbox utilities like .d-flex, .justify-content-*, and .align-items-* make it easy to build complex layouts without custom CSS.

Buttons and Icons

Use a custom CSS file to override default Bootstrap styles, keeping your changes separate and manageable.

Buttons and Icons

The .media component is perfect for aligning images and text side-by-side in blog posts, comments, or product listings.

Buttons and Icons

Use .bg-gradient with a background color class to create subtle gradient effects without custom CSS.

Buttons and Icons

Bootstrap’s form validation classes, like .was-validated, simplify handling user inputs with visual feedback.

Images

Bootstrap provides a set of utilities for handling images in a responsive and flexible way. With simple classes, you can make images responsive, apply rounded corners, create circular images, or even add borders for a thumbnail effect. These built-in classes help you quickly style and position images without the need for custom CSS, making image management seamless in your layouts.

image Styles

Bootstrap 5 offers 3 different table styles including rounded, circle and thumbnail:


<img src="#" class="rounded" alt="Cinque Terre">
<img src="#" class="rounded-circle" alt="Cinque Terre">
<img src="#" class="img-thumbnail" alt="Cinque Terre">

Create responsive images by adding an .img-fluid class to the <img> tag. The image will then scale nicely to the parent element.


<img class="img-fluid" src="ny.jpg" alt="New York">

Collapse

The Bootstrap Collapse component allows you to show and hide content with a smooth, collapsible transition. It can be used to create toggleable sections like FAQs, accordions, or navigation menus, helping to improve user experience by reducing clutter and making content more accessible only when needed. By combining collapse with buttons or links, you can easily control the visibility of elements on your webpage, creating dynamic, interactive user interfaces with minimal effort. Bootstrap’s built-in classes and attributes simplify the implementation, ensuring consistent and responsive design across devices.

Collapsibles

Collapsibles are useful when you want to hide and show large amount of content:

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. 

<button type="button" class="btn btn-primary" data-bs-toggle="collapse" data-bs-target="#demo">click me</button>
<div id="demo" class="collapse">
  Lorem ipsum dolor sit amet, consectetur adipisicing elit,
  sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam,
  quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</div>

The .collapse class indicates a collapsible element; this is the content that will be shown or hidden with a click of a button.

To control (show/hide) the collapsible content, add the data-bs-toggle="collapse" attribute to an a or a button element. Then add the data-bs-target="#id" attribute to connect the button with the collapsible content.

Cards

In Bootstrap, cards are a flexible and extensible content container with multiple variants and options. They replace old components like panels, wells, and thumbnails, providing a cohesive way to display content and UI elements. A Bootstrap card can contain headers, footers, images, links, buttons, lists, and more, making it suitable for a wide variety of use cases, such as showcasing products, presenting user profiles, displaying blog excerpts, or grouping related information.

Cards Styles

A card in Bootstrap 5 is a bordered box with some padding around its content. It includes options for headers, footers, content, colors, etc.

Modals

Bootstrap Modals are versatile, responsive components used to overlay content on top of the main page, grabbing the user’s attention for various interactions. They can display messages, forms, images, or custom HTML content in a sleek, customizable pop-up box. Modals offer a clean way to present important information or obtain user input without navigating away from the current page. With built-in accessibility and options to control their behavior, such as backdrop toggling, animations, and keyboard interactions, modals enhance user engagement and help create polished, interactive experiences on your website.

Modals

The Modal component is a dialog box/popup window that is displayed on top of the current page:

Modal Heading

Modal body.. 

  <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">
    Open modal
  </button>


<!-- The Modal -->

  <div class="modal-dialog">
    <div class="modal-content">

      <!-- Modal Header -->
      <div class="modal-header">
        <h4 class="modal-title">Modal Heading</h4>
        <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
      </div>

      <!-- Modal body -->
      <div class="modal-body">
        Modal body..
      </div>

      <!-- Modal footer -->
      <div class="modal-footer">
        <button type="button" class="btn btn-danger" data-bs-dismiss="modal">Close</button>
      </div>

    </div>
  </div>
</div>
