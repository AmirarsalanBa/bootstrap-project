Bootstrap Setup

Step-by-Step Guide

Step 1: Get Bootstrap

There are two ways to add Bootstrap to your project:

  • Use the CDN: The easiest way to start. It doesnâ€™t require downloading any files.
  • Download Bootstrap: If you want to use it offline or customize the files.

Step 2: Link Bootstrap

After downloading or using the CDN, you need to link the Bootstrap CSS and JavaScript files in your project:

  • CSS: Add the CSS link inside the <head> of your HTML file.
  • JavaScript: Place the JavaScript file at the end of the <body> to ensure proper loading.
<!-- Add this to the <head> for styles -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Add this to the end of the <body> for JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>

Step 3: Verify the Setup

Test if Bootstrap is working by using one of its components, such as a button:

<button class="btn btn-primary">Bootstrap Button</button>

Save your HTML file and open it in a browser. If the button looks styled, Bootstrap is set up correctly!
